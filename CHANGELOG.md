# Changelog

## 0.7.3 — Community Connectivity

- Fixed Bluetooth PAN activation race that could remove `bnep0`.
- Preserved existing PAN profiles instead of deleting them on every connection.
- Added PAN autoconnect with route metric 750.
- Added real route detection from the Linux routing table.
- Added guarded Wi-Fi radio control and automatic Wi-Fi recovery.
- Added a lightweight connectivity watchdog.
- Distinguished BlueZ connection from active PAN Internet.
- Improved diagnostics and corrected breakout detection.
