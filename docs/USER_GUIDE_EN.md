# CryptoGotchi v0.7.3 user guide

CryptoGotchi is a community market companion created by mikpoly for the Raspberry Pi Zero 2 W.

## Pages

- **Market**: assets, prices, charts, alerts and device state.
- **Companion**: mood, interactions, level, memory, achievements and journal.
- **Analysis**: a separate 15m, 1h and 4h technical view.
- **Cryptos**: manage the active watchlist.
- **Alerts**: dashboard sound and optional notification channels.
- **Settings**: Wi-Fi, Bluetooth, timezone, LCD and optional services.

The Analysis page displays trend, structure, support, resistance and a simulated buy/sell/wait companion bias. It is not financial advice.

The Bluetooth power switch changes Bluetooth only and never disables Wi-Fi or SSH.
